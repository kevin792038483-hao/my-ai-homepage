# 我的 AI 学习主页

这是我的第一个网页项目。

## 项目介绍

我是 AI 初学者，正在学习 ChatGPT、Codex、GitHub 和简单代码。

这个网页包括：

1. 一个大标题
2. 一段自我介绍
3. 一个学习目标列表
4. 一个可以点击的按钮

## 我学到了什么

通过这个项目，我学会了：

1. HTML 可以用来放网页内容
2. CSS 可以让网页变好看
3. JavaScript 可以让网页有互动
4. index.html 是网页文件
5. 用浏览器可以打开网页
6. 用记事本可以修改网页代码
